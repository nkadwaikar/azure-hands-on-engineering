# Microsoft Entra Backup & Recovery Track

This track covers Entra directory backup, difference reports, and object-level recovery using the Microsoft Entra admin center — portal-only.

## Track Structure

```text

`-- 1-Microsoft Entra Backup & Recovery.md
```

## Lab Sequence

1. [Microsoft Entra Backup & Recovery](1-Microsoft%20Entra%20Backup%20%26%20Recovery.md) — generate a difference report, perform an object-level recovery, and validate the restored state

## What it covers

- Microsoft Entra Backup & Recovery difference reports
- Object-level restore for supported cloud-managed objects (users, groups)
- Post-recovery validation steps
- Service behaviour and retention model overview

## Prerequisites

- Microsoft Entra tenant with **Global Administrator** or **Privileged Role Administrator** role
- Microsoft Entra ID P1 or P2 licensing
- Access to the [Microsoft Entra admin center](https://entra.microsoft.com)

> **Note:** Backup schedule and retention are Microsoft-managed. Tenant-level configuration of these controls is not available.
